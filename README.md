# 🎓 DDR Academy - Educational Website

## 📖 Project Overview

DDR Academy is a responsive educational website developed using HTML, CSS, and JavaScript. The website provides information about the academy, courses, campus gallery, student registration, and contact details. It is designed to be simple, responsive, and user-friendly.

---

## 🚀 Features

- 🏠 Home Page
- ℹ️ About Page
- 📚 Courses Page
- 🖼️ Gallery Page
- 📝 Student Registration Form
- 📞 Contact Page
- 📱 Responsive Design
- 🎨 Modern UI with CSS
- ⚡ JavaScript Form Validation
- 🖼️ Image Gallery
- 🔍 Basic SEO (Meta Title & Description)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

No additional software or installation is required.
